配信名：藤永勝也｜ひとりビジネスの教科書
No.5　2026年4月14日配信
タイトル：戻りたいのに戻れない夜の話
:::::::::::::::::::::::::::::

ある息子がいました。
「自分の人生を生きたい」と家を出て、
自由に生きる道を選びます。
ㅤ
ㅤ
最初はよかった。
でも、うまくいかない日が続き、
お金も尽きてしまう。
ㅤ
ㅤ
追い込まれた彼の胸に残ったのは、
ひとつの思いでした。
「帰りたい」
ㅤ
ㅤ
放蕩息子の話って、
聞いたことありますか。
今日は「戻りたいのに戻れない夜」の話です。
ㅤ
ㅤ
こんばんは。
藤永勝也です。
ㅤ
ㅤ
人生には、
「帰りたい場所」があるのに、
足が止まる時があります。
ㅤ
ㅤ
家族との関係。
昔の仲間。
本来の自分。
ㅤ
ㅤ
戻りたい気持ちはある。
でも、戻れない。
ㅤ
ㅤ
━━━━━━━━━━━━━
かっこ悪さが、いちばんの壁になる
━━━━━━━━━━━━━
ㅤ
ㅤ
帰りたくても帰れない。
戻りたくても、勝手に出てきた手前、戻れない。
かっこ悪くて戻れない。
ㅤ
ㅤ
「いまさら何て言えばいいのか」
「自分から離れたくせに、と言われそう」
「弱った姿を見せたくない」
ㅤ
ㅤ
だから人は、
本当は戻りたいのに、
平気なふりをしてしまう。
ㅤ
ㅤ
＝＝＝＝＝＝＝＝＝＝＝＝＝＝
人生を止めるのは失敗じゃない。
“見栄”と“恥”のほう。
＝＝＝＝＝＝＝＝＝＝＝＝＝＝
ㅤ
ㅤ
放蕩息子の話の核心は、
うまくやることじゃありません。
ㅤ
ㅤ
うまくやれなかった自分のまま、
それでも一歩戻ることです。
ㅤ
ㅤ
戻るって、
負けじゃない。
勇気です。
ㅤ
ㅤ
なぜなら、
戻るという行動は、
自分の見栄をいったん下ろす行為だからです。
ㅤ
ㅤ
人は、うまくいっている時より、
うまくいかなかった時のほうが、
本音がむき出しになります。
ㅤ
ㅤ
そこで「助けてほしい」と言えるか。
「やり直したい」と言えるか。
ここに、その人の強さが出ます。
ㅤ
ㅤ
本当に弱いのは、
傷つくのが怖くて、
平気なふりだけを続けることです。
ㅤ
ㅤ
本当に強いのは、
かっこ悪さを引き受けてでも、
もう一度つながろうとすること。
ㅤ
ㅤ
だから、
戻る人は弱い人ではなく、
人生を前に進める人なんです。
ㅤ
ㅤ
━━━━━━━━━━━━━
今日の小さな一歩
━━━━━━━━━━━━━
ㅤ
ㅤ
「本当は戻したいこと」を、
ひとつだけ書いてみてください。
ㅤ
ㅤ
そのあとに、
こう添えてみてください。
ㅤ
ㅤ
“かっこ悪くても、ここから戻る”
ㅤ
ㅤ
この一文が、
止まっていた人生を動かします。
ㅤ
ㅤ
今日も1日お疲れさまでした。
ゆっくりで大丈夫。
戻る勇気は、もう始まっています。
ㅤ
ㅤ
藤永勝也
ひとりビジネスプロデューサー

配信名：藤永勝也｜ひとりビジネスの教科書
No.5　2026年4月13日配信
タイトル：新しい手帳を開くみたいに、今日の一歩を置いてみる
::::::::::::::::::::::::::::

4月の半ば。
気持ちだけが先に
新しくなっていく季節ですね。
ㅤ
ㅤ
こんばんは。
藤永勝也です。
ㅤ
ㅤ
わたしはいま、
岐阜の各務原にいます。
ㅤ
ㅤ
朝晩はまだひんやりしますが、
昼間は歩くと汗ばむ日も出てきました。
ㅤ
ㅤ
新しい手帳を開く瞬間、
地球が静まるくらい神聖ですよね。
ㅤ
ㅤ
……3日後には、
貼り付けミスの残骸です。
（わかる人だけ、うなずいてOK）
ㅤ
ㅤ
さて、、、、
ㅤ
ㅤ
━━━━━━━━━━━━━
ひとりで仕事をつくる人へ
今日の話の置き場所
━━━━━━━━━━━━━
ㅤ
ㅤ
ここでお話ししているのは、
会社の中での評価の話ではなく、
ㅤ
ㅤ
「好きなこと」を軸に、
ひとりでビジネスを育てていく人の
現場の話です。
ㅤ
ㅤ
発信を始めたけれど反応が少ない。
勉強しているのに申し込みにつながらない。
ㅤ
ㅤ
そんなモヤモヤの正体は、
だいたいどこかに共通点があります。
ㅤ
ㅤ
それは、
「いきなり完璧を目指してしまう」
ことなんです。
ㅤ
ㅤ
＝＝＝＝＝＝＝＝＝＝＝＝＝＝
最初の一歩は、
宣言より「置き方」。
＝＝＝＝＝＝＝＝＝＝＝＝＝＝
ㅤ
ㅤ
新しい手帳を開いたとき、
いきなり一年分を埋めませんよね。
ㅤ
ㅤ
まずは今日の予定を
ひとつ書く。
ㅤ
ㅤ
ひとりビジネスも同じで、
最初に必要なのは才能の証明ではなく、
ㅤ
ㅤ
「続けられる置き方」
です。
ㅤ
ㅤ
━━━━━━━━━━━━━
今日やるのはこれだけでいい
━━━━━━━━━━━━━
ㅤ
ㅤ
ノートかスマホのメモに、
次の3つを書いてください。
ㅤ
ㅤ
1）誰の、どんな気持ちを軽くしたいか（ひとことで）
2）あなたが伝えたい「好き」は何か（ひとことで）
3）明日、誰かに見せてもいい形で残す場所（例：インスタ1枚、メモ1枚）
ㅤ
ㅤ
正解はいりません。
60点で提出で大丈夫です。
ㅤ
ㅤ
ここが置けると、
あなたの発信は
いきなり「作品」になります。
ㅤ
ㅤ
作品じゃなくて、
「今日のメモ」でいい。
ㅤ
ㅤ
その積み重ねが、
お客さまがあなたを見つける
手がかりになります。
ㅤ
ㅤ
今日も1日お疲れさまでした。
明日も、あなたのペースで一歩ずつ。
ㅤ
ㅤ
藤永勝也
