## PDFへの書き出し手順（Chrome）

1. `個別説明会後_検討資料.html` をChromeで開く（ダブルクリックでOK）
2. `Cmd + P` で印刷ダイアログを開く
3. 以下を設定：
   - **送信先**: 「PDFに保存」
   - **用紙サイズ**: A4
   - **倍率**: デフォルト（100%）
   - **余白**: 「なし」（HTML側で余白を組んでいるため）
   - **オプション**: 「背景のグラフィック」に**チェック**を入れる（重要・色が出ます）
   - **ヘッダーとフッター**: チェックを外す
4. 「保存」を押し、`個別説明会後_検討資料.pdf` として保存

## 編集のしかた

### 文章を直したいとき

`個別説明会後_検討資料.html` をテキストエディタ（Cursor／VSCode）で開き、HTMLタグの間にあるテキストを書き換えるだけ。

例：

```html
<h3>2ヶ月後</h3>
<p>
  手相の基本を一通り読めるようになり…  ← この文章だけ書き換える
</p>
```

### 金額を変更したいとき

15ページ目「お支払いパターン（全パターン）」の `<table class="price">` の中身を編集。

### 色味を変更したいとき

ファイル冒頭の `<style>` の `:root` ブロックを編集：

```css
:root {
  --c-bg: #FBF8F5;       /* 背景色 */
  --c-cream: #F4ECE2;    /* クリーム色（カード背景） */
  --c-rose: #D6A6A0;     /* ローズピンク（淡） */
  --c-rose-deep: #B8867E;/* ローズピンク（濃・タイトル色） */
  --c-gold: #C9A574;     /* ゴールド（飾り線） */
  --c-text: #3D3530;     /* 本文の文字色 */
}
```

## 運用メモ
