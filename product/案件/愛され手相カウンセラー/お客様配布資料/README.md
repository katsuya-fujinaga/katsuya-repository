# 愛され手相カウンセラー｜お客様配布資料

個別説明会後の保留者向けに配布する **ご案内資料（PDF）** の置き場所です。

## ファイル

| ファイル | 用途 |
|---|---|
| `愛され手相カウンセラー養成講座_ご案内資料.pdf` | **配布用（正）**。GitHub Pages のハブ（`index.html`）からもリンク |
| `index.html` | フォルダ内資料へのリンク一覧（Pages 用） |

## 正本と再生成

- **文章の正本**：`../個別相談/保留者向け講座まとめ資料.md`
- **PDF 生成**：`../個別相談/generate_summary_pdf.py` を実行すると、上記 PDF が上書きされる

```bash
python3 product/案件/愛され手相カウンセラー/個別相談/generate_summary_pdf.py
```

（実行環境・依存は同フォルダのスクリプト先頭コメントを参照）

## 運用メモ

- 個別説明会の最後に「家でじっくり見ていただける資料です」と PDF を共有
- メール添付時はファイルサイズを確認
- 公開 URL は `lp-deploy.mdc` の **`product/案件/.../お客様配布資料/`** ルールに従う
