# DAY1セミナー用｜開運クイズ 5本

## 九星覚醒カウンセラー｜本編中にそのまま使う用

> **このクイズの目的**
> - 参加者が**コメントしやすい空気**をつくる
> - DAY1の核である **「運は当てるだけでなく、動かすもの」** を体感してもらう
> - 九星気学とタロットの役割、開運行動の考え方を、**難しくなく理解してもらう**

---

## 使い方の前提

- すべて **3択** にしてあります
- 回答は **「1」「2」「3」だけでOK** にすると参加しやすいです
- 正解発表の前に、1人か2人へ **「それを選んだ理由もよかったら教えてください」** と聞くと盛り上がります
- 正解を責める空気ではなく、**「みんな途中までは合ってる。その中で今日はここを持ち帰ってくださいね」** で着地させるのがコツです

---

## 進行イメージ

### クイズを入れるおすすめ箇所

- クイズ①：冒頭フックの直後
- クイズ②：運の正体の話に入る前
- クイズ③：四つのつまずきの前後
- クイズ④：九星とタロットの役割分担に入るところ
- クイズ⑤：まとめ前

### 共通の振り方

```
ここで、ちょっとだけクイズです。
気軽に番号だけで答えてくださいね。
```

---

# クイズ①｜開運って、いちばん大事なのはどれ？

## ねらい

最初に「運＝場所やラッキーアイテムだけではない」と伝える。

## 出し方

```
Q.
開運したいとき、
いちばん大事なのはどれだと思いますか？

1. 吉方位やパワースポットに行くこと
2. 運気を待つこと
3. 自分に合う動き方を知って、小さく動くこと
```

## 正解

**3**

## 着地コメント案

もちろん、吉方位や場の力が悪いわけではありません。

でも、それだけだと現実は動きにくいんです。

大事なのは、**自分に合う流れを知って、いま何をするかまで落とすこと**。

今日のDAY1では、ここを一緒に整えていきます。

---

# クイズ②｜「来年よくなりますよ」と言われたら、その次に必要なのは？

## ねらい

「読むだけ」で終わらず、行動につなげる発想を入れる。

## 出し方

```
Q.
占いで
「来年は運気が上がりますよ」
と言われました。

このあと、いちばん欲しい言葉はどれですか？

1. じゃあ来年まで待ちましょう
2. 今いちばん気になっていることは何ですか？
3. とにかく大丈夫です
```

## 正解

**2**

## 着地コメント案

未来の見通しがあるのは大事です。

でも、人はそれだけでは動けないんですね。

**「で、今なにをしたらいいの？」** が見えたときに、初めて動きやすくなります。

だから九星覚醒では、読んで終わりではなく、**次の一手**まで大事にします。

---

# クイズ③｜「私、運が悪いんです」と言う人に、まず持ちたい視点は？

## ねらい

「運が悪い」の裏にある、リソース分散という見方をやさしく入れる。

## 出し方

```
Q.
「私、運が悪いんです」
と何度も言う人がいます。

でもよく聞くと、
勉強も、副業も、人間関係も、開運も、
全部がんばっていて疲れています。

このとき、まず持ちたい視点はどれ？

1. この人は本当に運が悪い
2. 才能がないのかもしれない
3. 力が散っていて、確率が薄くなっているのかもしれない
```

## 正解

**3**

## 着地コメント案

ここ、すごく大事です。

「運が悪い」と決めつけると、自分を責めやすくなります。

でも実際は、**力がないのではなく、散っているだけ**ということがあるんです。

そう見えると、必要なのは自己否定ではなく、**順番を整えること**だとわかってきます。

---

# クイズ④｜転職の相談。最初の一手として近いのはどれ？

## ねらい

九星とタロットの前に、まず「何が欠けているかを見る」感覚を入れる。

## 出し方

```
Q.
「転職したいけど動けません」
という相談が来ました。

最初の一手として、いちばん近いのはどれ？

1. すぐに未来の運気を話す
2. すぐにカードを引く
3. いちばん怖いことは何ですか？と一言聞く
```

## 正解

**3**

## 着地コメント案

これ、九星がダメとか、タロットがダメという話ではありません。

大事なのは、**その人が今どこで止まっているか**を見ることです。

未来が見えなくて止まっているのか。
気持ちが言葉にならなくて止まっているのか。

そこが見えると、九星から入るのか、タロットから入るのかも選びやすくなります。

---

# クイズ⑤｜一般論で終わらせないために、ひとつ足すなら？

## ねらい

「その人の話」に変える一言の大切さを伝える。

## 出し方

```
Q.
「一白水星の人はこうですね」
と一般論だけで終わると、
相手が少し冷めてしまうことがあります。

では、ひとつ足すなら何がよさそうですか？

1. もっと専門用語を増やす
2. あなたの場合、今いちばん気になっているのはどこですか？と聞く
3. 他の星の説明も追加する
```

## 正解

**2**

## 着地コメント案

相手が欲しいのは、教科書の説明だけではないんです。

**「私の話としてわかってもらえた」**
この感覚があると、占いの言葉が急に入っていきます。

九星は材料です。
そこに **相手の今** を重ねた瞬間、鑑定がぐっと生きてきます。

---

## 講師向けメモ

### 盛り上げ方

- 「直感でいいですよ。深く考えなくて大丈夫です」と先に言う
- コメントが止まったら「見る専の方は、心の中で1つ決めるだけでも大丈夫です」と逃げ道をつくる
- 正解発表の前に「けっこう割れましたね」「これ多いですね」と一言はさむ

### 教育として外さないポイント

- クイズの正解そのものより、**なぜその答えになるのか** を短く伝える
- 毎回の着地は、次のどれかに必ずつなぐ
  - 運は当てるだけでなく動かす
  - 未来と感情の両方を見る
  - 最後は一手に落とす

### トーン

- 「間違いです」ではなく **「今日のDAY1で持ち帰ってほしいのはここです」**
- 参加者を試す空気ではなく、**一緒に感覚を育てる空気** にする
