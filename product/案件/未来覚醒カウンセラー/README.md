# 未来覚醒カウンセラー｜ドキュメント索引

九星気学 × タロット／電話・チャット占い師養成（無料 **4日間LIVE**）まわりの置き場。LP・広告・企画ナレッジ・台本は**フォルダ別**で管理する。

## 📌 概要

- **クライアント**：本田有紀華さん（占い／カウンセラー系ブランド）。案件横断ナレッジは `product/講師横断/本田有紀華さん案件/`。
- **目的**：電話・チャット占い師を視野に、「開運を当てる」ではなく「**動かす**」コンセプトでフロントを回すこと。
- **現状**：企画／コンセプトは **マスター** に集中。DAY1 シナリオ・レター LP ドラフトまであり。**DAY2〜4 シナリオ**はマスターの設計のみ（台本ファイルは未）。
