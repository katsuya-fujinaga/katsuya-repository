# 本田有紀華｜全案件共通ハブ

本田有紀華さん関連の**全案件共通**で使う正本を集約するフォルダです。

## 目的

- 案件ごとに重複しやすい「共通パーツ」を一本化する
- 修正時の更新漏れを防ぐ
- Cursor ルールと実ファイルの参照先を一致させる

## 収録ファイル

| ファイル | 用途 |
|----------|------|
| `00_全案件共通ルール.md` | 3ブランド共通で守る運用原則（表記・導線・共通判断） |
| `honda-yukika-mail-footer.md` | MyASP用フッターの正本（3ブランド分） |

## 対象ブランド

- やさしく売れる占い師
- 星使いカウンセラー
- 未来覚醒カウンセラー
- （必要に応じて今後追加）

## 運用ルール

1. **共通変更はこのフォルダを先に更新**する。
2. 案件側に例外がある場合は、案件フォルダに「例外理由」を明記する。
3. Cursor ルール（`.cursor/rules/honda-yukika-*.mdc`）は、このフォルダの正本に合わせる。
