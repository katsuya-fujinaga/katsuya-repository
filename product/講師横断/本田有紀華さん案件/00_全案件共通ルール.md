# 本田有紀華｜全案件共通ルール

案件をまたいで共通化するための運用メモ。案件固有の例外があるときは案件側 README に追記する。

## 1. 表記・名義

- 氏名表記は **本田有紀華** で統一
- 差出人名は原則 **本田有紀華**
- 3ブランド共通SNSリンクは `honda-yukika-mail-footer.md` の正本を使用

## 2. フッター運用

- フッター正本：`honda-yukika-mail-footer.md`
- 改行・空行（ㅤ）など形式は `.cursor/rules/email-writing-format.mdc` に合わせる
- ブランドごとの差分は「本文上部」または「案件側メタ」で吸収し、フッター正本は可能な限り共通維持

## 3. メール本文の共通判断（骨格）

- 形式（件名、冒頭メタ、改行、構成順）は `.cursor/rules/email-writing-format.mdc`
- トーン・語り口・構成思想は `.cursor/rules/honda-yukika-housemail.mdc`
- 本ファイルは「案件横断でズレやすい共通項」の確認用

## 4. 案件で例外を作るとき

- 例外は最小にする
- 例外理由を案件 README に1行残す
- 期間限定例外は終了日を明記する
