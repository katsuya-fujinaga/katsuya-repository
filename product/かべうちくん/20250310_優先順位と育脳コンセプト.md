## 育脳5つのお教室 新コンセプト

### キーコンセプト

> **「点ではなく線で育脳する」**

---

### 問題提起

今の市場は「点」しかない。

| サービス | 問題 |
|----------|------|
| ベビーマッサージだけ | 点で終わる |
| リトミックだけ | 点で終わる |
| 幼児教室だけ | 点で終わる |

**→ 部分的にしかやっていないから賢い子に育たない**

---

### 解決策

0歳〜6歳を一貫して「線」でつなぐ

```
0歳 → 1歳 → 2歳 → 3歳 → 4歳 → 5歳 → 6歳
 │      │      │      │      │      │      │
 └──────┴──────┴──────┴──────┴──────┴──────┘
              ひとつの「線」でつなぐ
```

- 年齢に合わせた5つのお教室
- 一人のカウンセラーが継続サポート
- 脳の発達段階に合わせた働きかけ

---

### このコンセプトの強み

1. **差別化が明確**
   「ベビーマッサージだけ」「幼児教室だけ」とは全然違う

2. **継続課金の理由になる**
   0歳から6歳まで＝最大6年間のサポート

3. **口コミが生まれやすい**
   「この先生に任せれば安心」という紹介

4. **カウンセラーの価値が上がる**
   単なるお教室の先生ではなく「育脳の専門家」
