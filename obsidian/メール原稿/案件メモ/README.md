# 案件メモ

案件ごとのメール下書き・推敲メモ用です。

## 使い方

1. **`product/案件/`** のフォルダ名と揃えたサブフォルダをここに作成する（例：`育脳子育てカウンセラー/`）。  
2. その中に `.md` や `.txt` で下書きを置く。  
3. 配信確定文を Git に残す場合は、案件の **`product/案件/<案件名>/メール/`** など、案件側の正規フォルダへ移す（任意）。

全案件共通の**形式**は **`.cursor/rules/email-writing-format.mdc`**。トーン・フッターは案件・ブランドの Cursor ルールを参照してください。
