# X｜アカウント名の候補と裏テーマ

**用途**：表示名・固定文で迷ったときに読む**内部用**。表に書ききれない「意図」をここに固定する。

**関連**：`Xブランディング_共感・バズ・保存_ナレッジ.md`

---

## 1. 表示名の候補（2案）

| 表示名 | 表で伝わること | 向く読者 |
|--------|----------------|----------|
| **藤永勝也｜50代からの商品の届け方** | **仕事・プロ**としての「届け方」。講座・サービス・オファーまで含むニュアンスが出やすい。 | ひとりビジネスで**売上・顧客**まで視界に置いている人。保存・権威寄り。 |
| **藤永勝也｜50代からの理想の届け方** | **人生・ライフクリエイト**に寄せた「届け方」。広めに「自分らしい生き方」まで含めて読める。 | いまは悩み・共感・理想の言語化に寄る読者。フォロー・共感寄り。 |

**使い分けの目安**

- **商品** … 「何を売るか・どう届けるか」が名前から連想されやすい。**ビジネス寄りのブランド**になりやすい。
- **理想** … 「ライフクリエイト」と**語感が近い**。**詩的に広がるぶん、プロフで具体を補うと安心**（届け方＝設計・仕組み・講座の道筋、と一言足すなど）。

---

## 2. 裏テーマ（公開文にフルでは書かない核）

**一言**：

> **「届け方」で、50代の人生と仕事の両方を、自分の手で形にする（ライフクリエイト）。**

**内側で必ず揃える3点**（どちらの表示名でも同じ）：

1. **誰の物語か** … 50代・再起・「好き」を仕事にしたい人（無理に若者向けにしない）。
2. **何のプロか** … マーケター名乗りより**設計屋・裏方プロデューサー**（順番・導線・言葉の置き場）。
3. **何を売っていないか** … キラキラ起業、煽り、**テクだけ**。失敗と現場感は**資産として出す**。

**「届け方」の定義（内部）**：

- 表層：LP・導線・説明の順番・売り込み感の整え方。
- 深層：**自分の理想（ライフクリエイト）が、他者に伝わる形になる**までの設計。  
  → **理想の届け方**という名前を選んでも、実務の中身は「商品の届け方」と**矛盾しない**（同じ設計思考でつなぐ）。

---

## 3. テーマ語との対応

| 表の言葉 | 裏で固定している意味 |
|----------|----------------------|
| ライフクリエイト | 人生を作り直す意志×**届け方の技術**（精神論だけにしない）。 |
| 50代から | 再出発・キャリア後半の**現実**（時間・体力・家庭を置き去りにしない）。 |
| 届け方 | 見せ方・順番・共感の言語化・**誰が言うか**まで含む。 |

---

## 4. 表示名を決めたあとのチェック

- [ ] プロフィール2行目に、**「ひとりビジネス／設計」「ライフクリエイト」**のどちらかが入っているか。
- [ ] **理想**を名前にした場合：**具体的な職能**（プロデューサー・設計）をプロフで担保しているか。
- [ ] **商品**を名前にした場合：**人生の共感**が投稿の①④で定期的に出ているか（冷たいノウハウ垢になっていないか）。
