# ひとびじ（藤永勝也｜個人プロジェクト）

Katsuya Fujinaga 個人ブランド **ひとびじLAB** まわりの原稿・ナレッジを、このフォルダに集約しています。

**Git 上の正本はこのフォルダ（`katsuya_project/ひとびじ/`）**です。同じ **Katsuya プロジェクト**の **型**（`セールスマインドセット/`）はひとつ上の階層に並びます。Obsidian ボルトでは **`obsidian/ひとびじ`** がここへのシンボリックリンクになります。

| サブフォルダ | 内容 |
|--------------|------|
| **`LAB/`** | ブランド設計・X プロフィール・ペルソナ・マインドセット・運用メモなど（**ひとびじLAB の本文はすべてここ**）。 |
| **`ハウスメール/`** | メルマガ「ひとりビジネスの教科書」原稿。 |
| **`X/`** | X 投稿ストック（`Xツイート文例.md`）、非集客フェーズの戦略メモ（`X_非集客フェーズ_戦略メモ.md`）。運用設計の正本は **`LAB/X運用_濃いフォロワー設計_非集客フェーズ.md`**。 |

メール運用のインデックス（全案件の形式・ボルト・`product/` の対応）は **`../メールライティング/README.md`**。

Cursor のブランド正本は引き続き **`.cursor/rules/hitobizi-lab.mdc`**。ワークスペース上のコピー・たたき台は **`LAB/`** を編集してください。
