口座名（最大8件）

    固定費を一括入力

    指定した期間の毎月同じ日に、引落または入金をまとめて登録します。

    借入残高の設定

    「項目キーワード」を項目メモに含めて入力すると、引落累計に自動反映されます。

    CSV出力（期間指定）

    指定した開始月〜終了月の取引をCSVにまとめます。口座（銀行）ごとに列ブロック（引落・入金・項目・残高）が横に並び、1行＝その日のその「行」です。Excel向け UTF-8（BOM付き）。

    Googleドライブ連携

    OAuth Client ID を 1 度だけ入力すると、このブラウザから Drive API で cash-ledger-sync.json を読み書きできます。 手順は README.md（このフォルダ直下）を参照してください。

    スコープは drive.file のみです（このアプリが作成・選択したファイル以外にはアクセスしません）。
